# Pacotes utilizados

## Nuxt Cron

É um pacote simples para a criação de Cron Jobs, permitindo executar código periodicamente no servidor.
Toos os cron jobs devem ser adicionados dentro da pasta /server/cron/<novo-arquivo-cron>

Referência: https://nuxt-cron.hywax.space/examples/async.html

## Supabase

## Módulo Supabase nuxtjs

É um módulo do supabase para trabalhar exclusivamente com o Nuxt. Serve justamente para identificarmos como comunicar com o Supabase.
As demais configurações são feitas usando a documentação do Supabase mesmo.

https://supabase.nuxtjs.org/services/serversupabaseclient

## TWILIO

Módulo do Twilio para enviar mensagens pelo whatsapp. Somente em teste por enquanto.
